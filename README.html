ARM Microcontroller Frameworks

Here are some firmware development frameworks for certain 32-bit ARM (Advanced RISC Machine) single chip microcontrollers. I began in 2006 with my own somewhat vendor independent GCC C/C++ development framework. I am interested in alternatives to the traditional GCC development environment, so there are also some experiments with Alire Ada, Arduino C++, Astrobe Oberon, and Mikropascal for ARM microcontrollers.
