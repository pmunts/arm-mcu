# ARM Microcontroller Frameworks

Here are some firmware development frameworks for certain 32-bit
<a href="http://www.arm.com" target="_blank">ARM</a> (Advanced RISC
Machine) single chip microcontrollers. I began in 2006 with my own
somewhat vendor independent [GCC C/C++](https://gcc.gnu.org) development
framework. I am interested in alternatives to the traditional GCC
development environment, so there are also some experiments with [Alire
Ada](https://pico-doc.synack.me), [Arduino
C++](https://www.arduino.cc/), [Astrobe Oberon](http://www.astrobe.com),
and [mikroPascal](https://www.mikroe.com/mikropascal-arm) for ARM
microcontrollers.
