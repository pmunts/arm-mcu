With the Arduino-Pico core, if FreeRTOS is enabled (:os=freertos appended
to the FQBN), the normal Arduino main program function main() that calls
setup() and loop() is replaced by a FreeRTOS specific main() that creates
some background tasks and then calls vTaskStartScheduler().

The setup() and setup1() functions are called by tasks running on CPU cores
0 and 1 respectively.  The setup1() function is optional.

The loop() and loop1() functions are called by idle tasks running on CPU cores
0 and 1 respectively.  Both loop() and loop1() functions are optional.

The FreeRTOS specific main() also creates a task that provides USB serial
port emulation.  Serial.println() et al are redirected to the USB serial port.
